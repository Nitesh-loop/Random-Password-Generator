# Random Password Generator
A simple web-based password generator built with HTML, CSS, and JavaScript. Generate secure random passwords with customizable lengths and options.

## Features
- Generate **two passwords at once**
- Customize password length using a **range slider**
- Optional: Copy passwords to clipboard on click
- Optional: Toggle symbols and numbers on/off
- Responsive and user-friendly design

## How to Use
1. Open `index.html` in a web browser.
2. Click the **Generate Password** button.
3. Adjust the **length slider** to change password length.
5. Enjoy your secure passwords!
